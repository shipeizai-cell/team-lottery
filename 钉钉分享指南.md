# 钉钉分享使用指南

## 🚀 快速开始：5分钟部署

### 方案一：使用模拟版（最简单）
1. 将 `lottery-advanced.html` 上传到公司共享文件夹
2. 获取可访问的URL（如 `http://公司文件服务器/抽奖.html`）
3. 复制URL到钉钉群

### 方案二：使用Supabase版（推荐）
1. 按照《Supabase设置教程.md》配置数据库
2. 部署 `lottery-supabase-ready.html` 到服务器
3. 获取公开访问URL
4. 分享到钉钉

## 🔗 生成钉钉可点击链接

### 方法1：直接发送URL
钉钉会自动识别HTTP链接，用户可直接点击。

### 方法2：使用链接卡片
```
[团建投票系统](https://你的链接.com)
```

### 方法3：创建钉钉机器人（高级）
1. 在钉钉群设置中添加自定义机器人
2. 获取Webhook地址
3. 发送带按钮的卡片消息

## 📊 实时监控和管理

2. **监控参与情况**：查看谁已投票，谁还没投
3. **导出数据**：投票结束后导出CSV文件

## ⚙️ 系统配置建议

### 投票截止时间设置
在代码中修改：
```javascript
// 设置投票截止时间
this.voteEndTime = new Date('2025-03-24T20:00:00');
```

### 团队人数设置
```javascript
// 设置团队总人数
this.maxParticipants = 50;
```

### 防刷票设置
- 每个用户ID只能投票一次
- 记录IP地址防止重复投票
- 可设置同一IP最大投票次数

## 🎯 使用流程

### 组织者流程：
1. **前期准备**（投票前1天）
   - 确定投票选项
   - 部署投票系统
   - 测试系统功能

2. **启动投票**（当天）
   - 在钉钉群发布投票通知
   - 说明投票规则和截止时间
   - 回答同事问题

3. **过程监控**（投票期间）
   - 监控投票进度
   - 发送催投提醒（如需）
   - 解答技术问题

4. **公布结果**（投票截止后）
   - 截图最终统计结果
   - 在钉钉群公布
   - 安排后续团建事宜

### 参与者流程：
1. 点击钉钉中的投票链接
2. 查看选项说明
3. 点击"开始抽奖"按钮
4. 查看自己的投票结果
5. 观察实时统计变化

## 🔧 故障排除

### 常见问题1：链接打不开
✅ 解决方案：
- 检查URL是否正确
- 确认服务器可访问
- 尝试用手机浏览器打开测试

### 常见问题2：无法投票
✅ 解决方案：
- 刷新页面重试
- 检查是否已投过票
- 换个浏览器尝试

### 常见问题3：统计不更新
✅ 解决方案：
- 等待几秒自动刷新
- 手动刷新页面
- 检查网络连接

### 常见问题4：同事反映问题
✅ 标准回复模板：
```
感谢反馈！请尝试：
1. 刷新页面
2. 换个浏览器
3. 清除缓存后重试

如仍无法解决，请提供：
- 错误截图
- 浏览器类型
- 操作步骤

我们会尽快处理！
```

## 📈 最佳实践

### 投票时间安排
- **建议时长**：4-8小时
- **最佳时段**：工作日上午10点启动，下午6点截止
- **避免时段**：午休时间、下班前后

### 提高参与率
1. **多次提醒**：开始、中期、截止前各提醒一次
2. **简化操作**：确保投票流程简单直观
3. **明确价值**：说明投票的重要性和影响

### 结果公布
投票结束后，在钉钉公布：
```

🏠 家里蹲：XX票 (XX%)
🍽️ 歆弘府：XX票 (XX%)

🏆 最终决定：XXX

共有XX位同事参与投票，参与率XX%。

接下来我们将根据投票结果安排团建事宜，具体安排另行通知。

再次感谢大家的参与！🎊
```

## 🆘 紧急情况处理

### 系统崩溃
1. 立即切换到备用方案（如微信接龙）
2. 在钉钉说明情况
3. 尽快修复系统

### 数据异常
1. 检查数据库记录
2. 核实异常投票
3. 必要时重置投票

### 争议处理
1. 公布原始数据
2. 解释投票规则
3. 必要时重新投票
