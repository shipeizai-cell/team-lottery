创建Supabase项目

Supabase是一个免费的PostgreSQL数据库服务。请按以下步骤操作：

操作步骤

请在新标签页中完成这些操作

步骤1: 访问Supabase
打开Supabase网站
点击上方按钮在新窗口打开Supabase，然后继续下面的步骤
步骤2: 注册并创建项目
  1. 点击 "Start your project"
  2. 使用GitHub或邮箱注册
  3. 创建新组织（输入组织名称）
  4. 创建新项目：
    • Name: team-lottery-system
    • Database Password: 设置强密码（记住它！）
    • Region: 选择 Asia Southeast (Singapore)
    • Pricing Plan: 选择 Free Plan
  5. 点击 "Create new project"（等待1-2分钟）
3
获取API凭证

现在需要获取Supabase项目的连接信息：

在Supabase中操作
  1. 项目创建完成后，进入项目仪表板
  2. 点击左侧菜单 Settings → API
  3. 找到以下信息并填写到下方：
以 https:// 开头，结尾是 .supabase.co
以 eyJ 开头的长字符串，在 "Project API keys" 部分
可以在URL中找到，或在 "General settings" 中查看

重要提示

请妥善保存这些凭证，不要分享给他人

4
自动配置数据库

系统将自动创建数据库表和安全策略：

-- 创建参与者表
CREATE TABLE participants (
  id BIGSERIAL PRIMARY KEY,
  user_id TEXT UNIQUE NOT NULL,
  user_name TEXT NOT NULL,
  choice TEXT NOT NULL,
  user_ip TEXT,
  user_agent TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);

-- 启用安全策略
ALTER TABLE participants ENABLE ROW LEVEL SECURITY;

-- 允许所有人插入和读取
CREATE POLICY "allow_insert" ON participants
FOR INSERT WITH CHECK (true);

CREATE POLICY "allow_select" ON participants
FOR SELECT USING (true);

准备配置数据库

点击下方按钮开始自动配置
